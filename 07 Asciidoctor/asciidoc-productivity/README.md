# AsciiDoc Productivity
Hilfreiche Snippets für AsciiDoc in VS Code.

## Installation der Umgebung

```
npm install -g yo generator-code
npm install -g @vscode/vsce
```

## Erzeugen des Packages

```
vsce package --allow-missing-repository
```

Es entsteht eine vsix Datei, die unter Extensions -> ... -> Install from VSIX geladen werden kann.
